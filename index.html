Projects

Khol-s-website Clone

The Kohl's website clone is a comprehensive project built using React and Chakra UI and JS , showcasing advanced frontend and backend capabilities. It features secure user authentication and dynamic routing for smooth navigation. The real-time password feedback enhances user experience during account creation. Users can easily add items to their cart and delete them, with changes reflected instantly thanks to seamless integration with Axios. The project simulates a backend using JSON Server, deployed on Render, and efficiently manages user data with LocalStorage.

Tech Stack: HTML5, CSS3, JavaScript, React, Chakra UI, JSON Server, Axios, LocalStorage

bigbasketclone

The BigBasket Clone project is a fully responsive e-commerce website designed to offer an optimal user experience across all devices, from mobile phones to desktops. It features advanced sorting functionality, allowing users to sort products by price, rating, and discount in both ascending and descending orders. A robust search feature enables users to find products quickly from the product page. Product data is dynamically retrieved from a JSON file, simulating a realistic backend environment. The project includes user authentication capabilities, allowing users to sign up, log in, and add products to their cart, with secure redirection to the login page if they attempt to access the cart without logging in. LocalStorage is used to store user data and cart information, ensuring persistence and quick access.

Tech Stack: HTML5, CSS3, JavaScript, Fetch API, LocalStorage
